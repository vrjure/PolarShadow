# PolarShadow

一个可以自定义源的网络资源获取工具。

## 原理

利用jsonpath/xpath将获取到的内容转换成合适的json。

## UI

AvaloniaUI

## 功能

- [ ] 自定义源
- [ ] 资源搜索
- [ ] 资源浏览
- [ ] 在线播放
- [ ] 集成aria2RPC
